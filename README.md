# Monomorphic Angular 2 seed

### common steps
```sh
tns install
```

### Nativescript start
```sh
npm install -g nativescript (run only once)
tns emulate ios
tns emulate android
```
### Webapp start
```sh
npm start
```
open (http://localhost:9000/)

### Server start
```sh
npm run server
```

### Webapp and server start 
```sh
npm run all
```
